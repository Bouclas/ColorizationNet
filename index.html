TensorFlow.js implementation of an encoder-decoder neural network trained for the task of photo colorization.

The architecture is a three-branch architecture, with three encoder-decoder sub-nets, each having the task of predicting the value for each of the 3 color channels (R, G, B)! The specific architecture has been trained on a small set of personal photos (around 300 which are augmented with just zoom fluctuations)! The entire training has been performed in Google Colaboratory! If you are interested in the code, feel free to contact me!
